# Разработка автоматизированной среды анализа причин возникновения нежелательных событий в КФС на основе байесовской сети

## Описание проекта

Данный проект представляет собой программное решение для анализа причин возникновения неисправностей в киберфизических системах (КФС) с использованием вероятностных байесовских сетей.

Программа позволяет:
- строить причинно-следственные модели;
- выполнять вероятностные расчёты;
- выявлять корневые причины нежелательных событий;
- визуализировать структуру системы.


## Используемый подход

В основе решения лежит **причинно-следственная байесовская сеть**, которая:
- представляется в виде направленного ацикличного графа;
- моделирует вероятностные зависимости между событиями;
- позволяет выполнять различные типы вывода:
  - априорный;
  - прямой;
  - обратный (диагностика);
  - смешанный.


## Архитектура приложения

Приложение реализовано по шаблону **MVC (Model-View-Controller)**:

- **Model** — хранение данных и логика работы с байесовской сетью;
- **View** — графический интерфейс пользователя;
- **Controller** — обработка действий пользователя и управление приложением.


## Технологии

- **Java**
- **JavaFX** — графический интерфейс
- **FXML** — декларативная разметка UI
- **Spring Framework** — внедрение зависимостей
- **Maven** — управление зависимостями и сборка
- **Jlink / Jdeps** — создание минимального runtime
- **Launch4j** — упаковка в `.exe`


## Основной функционал

### Работа с байесовской сетью
- Создание узлов и связей
- Определение таблиц вероятностей
- Построение полной таблицы распределения
- Выполнение вероятностных вычислений

### Типы вычислений
- Маргинальные вероятности
- Апостериорные вероятности
- Диагностика (обратный вывод)

### Работа с файлами
- Сохранение модели
- Загрузка модели
- Проверка корректности данных

### Интерфейс
- Визуальное построение сети
- Отображение вероятностей
- Выделение причин и следствий

## Структура проекта

Основные компоненты:

- `ModelNode` — узел сети
- `AppModel` — модель всей сети
- `MainController` — логика управления
- `NodePane / LogicalPane` — визуальные компоненты
- `MyMath` — вспомогательные вычисления


## Пример применения

Программа позволяет моделировать такие ситуации, как:
- неисправности датчиков;
- перегрузка вычислительных систем;
- ошибки программного обеспечения;
- сбои передачи данных.

И находить их вероятные причины.
